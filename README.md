# 気づかずエシカル (MVP V2.3)

「世界を良くしたいという願い」を、日々の買い物という「行動」へ自然に繋げる。

## 🌟 コンセプト
エシカルとは、世界を思いやる選択のこと。
例えば「戦争のない平和な世界」を願っていても、日々の買い物がそこにどう繋がっているかを見極めるのは簡単ではありません。
このサービスは、あなたの純粋な願いと消費活動を自然に結びつけ、気づかないうちに心地よい選択ができるようサポートします。

## 🎨 デザイン・アイデンティティ (Pinterest Aesthetic)
- **Sophisticated Minimal**: セージグリーンとオフホワイトを基調とした、目に優しくプレミアムな質感。
- **Glassmorphism**: 情報を美しく凝縮する背景ぼかしカードUI。
- **Modern Typography**: `Outfit` と `Inter` による、クリーンで意思のあるタイポグラフィ。

## ⚙️ 主な機能
1. **多角的パーソナライズ**: 10のライフスタイル、50の趣味データを統合したAIマッチング。
2. **3つの価値観（感度）設定**: 
   - 戦争・政治への加担（軍事、占領地支援など）
   - 人権侵害・労働問題（児童労働、差別、性的加害者擁護など）
   - 環境破壊・動物搾取（汚染、気候変動、動物実験など）
3. **消耗品特化型リスト**: 米、コーヒー、洗剤など、日常的に繰り返し購入するアイテムに特化。
4. **エシカル・ダイレクトリンク**: 生産者直販、フェアトレード認証ストアなど、信頼できる販路へ最短距離で誘導。
